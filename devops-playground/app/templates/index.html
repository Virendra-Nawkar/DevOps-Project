🧰 Kubernetes Quick Commands

Scaling
Watch HPA live: kubectl get hpa -n devops -w Manual scale: kubectl scale deploy devops-playground --replicas=5 -n devops View pod events: kubectl describe pod -n devops -l app=devops-playground
Logs & Debugging
Tail logs all pods: kubectl logs -l app=devops-playground -n devops -f Exec into pod: kubectl exec -it <pod> -n devops -- bash Top pods: kubectl top pods -n devops
Rolling Updates
Update image: kubectl set image deploy/devops-playground app=devops-playground:v2 -n devops Rollback: kubectl rollout undo deploy/devops-playground -n devops Status: kubectl rollout status deploy/devops-playground -n devops
